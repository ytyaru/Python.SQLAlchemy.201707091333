# このソフトウェアについて

SQLAlchemyのチュートリアルをやってみた。

# 開発環境

* Linux Mint 17.3 MATE 32bit
* Python 3.6.1
* [SQLite](https://www.sqlite.org/) 3.8.2
* SQLAlchemy 1.1.9
